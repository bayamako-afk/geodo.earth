# クラウド契約管理システム (SPFx Web Part)

SharePoint Framework (SPFx) 1.20 / React 17 で構築したクラウドサービス契約管理ウェブパートです。

## バージョン

| バージョン | ファイル | 説明 |
| :--- | :--- | :--- |
| v1.0.0 | `cloud-contract-manager.sppkg` | 第1フェーズ（通知・BIなし） |

## 機能概要

### 第1フェーズ（v1.0.0）

- **クラウド契約マスタ管理**: サービス名・ベンダー・カテゴリ・利用部署・有効期限・金額などを一覧管理
- **ライセンス利用者管理**: 各サービスのアカウント・ライセンス種別・利用者を管理
- **契約更新・精査管理**: 更新判断（継続/解約/プラン変更）の記録と進捗管理
- **有効期限の残日数表示**: 30日以内（赤）・60日以内（黄）・期限超過（赤太字）で色分け表示
- **SharePointリスト自動作成**: 管理者が「リスト初期化」ボタンをクリックするだけで3リストを自動作成

### 第2フェーズ以降（予定）

- Power Automate による有効期限リマインダー通知（60日前・30日前）
- Power BI によるコスト可視化ダッシュボード

## SharePointリスト構成

| リスト名 | 用途 |
| :--- | :--- |
| `Cloud_Contract_Master` | クラウド契約マスタ（サービス・金額・有効期限等） |
| `Cloud_License_Users` | ライセンス利用者管理 |
| `Cloud_Contract_Review` | 契約更新・精査記録 |

## インストール手順

1. SharePoint 管理センター > アプリカタログ を開く
2. `cloud-contract-manager.sppkg` をアップロードしてデプロイ
3. 対象サイトでアプリを追加
4. ページにウェブパートを追加
5. ウェブパートのプロパティパネルで管理者メールアドレスを設定
6. 「リスト初期化」ボタンをクリックして SharePoint リストを自動作成
