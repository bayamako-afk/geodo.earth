# 職場ウイルス図鑑 制作ルール

本ドキュメントは「職場ウイルス図鑑」シリーズの映像・音声・テキストコンテンツ制作において共通して適用するルールをまとめたものである。

---

## 1. タイトル表記ルール

### 1-1. 画面表示・テキスト表記

| 項目 | 表記 |
|------|------|
| シリーズタイトル | 職場ウイルス図鑑 |
| ファイル番号付き | 職場ウイルス図鑑 File.XX |
| サブタイトル例 | 職場ウイルス図鑑 File.01 フキゲン・ウイルス |

### 1-2. ナレーション・読み上げ用テキスト

**漢字の自動読みに任せず、読み上げ用テキストは必ずひらがなで明示すること。**

| 表示テキスト | 読み上げ用テキスト（ひらがな） |
|------------|--------------------------|
| 職場ウイルス図鑑 | しょくばウイルスずかん |
| 職場ウイルス図鑑 File.01 | しょくばウイルスずかん ファイル ゼロいち |
| フキゲン・ウイルス | フキゲン・ウイルス |

> **理由：** 「職場」は「しょくば」と読むが、TTS（テキスト音声合成）エンジンや音声モデルが「しょくじょう」「しょくち」等の誤読をする可能性があるため、読み上げ用テキストは常にひらがなで明示する。

---

## 2. 音声・ナレーション制作ルール

- TTS（テキスト音声合成）を使用する場合、読み上げ原稿はひらがな・カタカナ混じりの「よみがな原稿」を別途用意すること
- 固有名詞・造語（例：フキゲン・ウイルス、ため息ウイルス等）は初出時に読み確認を行うこと
- 数字・英数字の読み方も明示すること（例：`File.01` → `ファイル ゼロいち`）

---

## 3. 映像制作ルール（Zopia CLI）

### 3-1. スタイル選定

| 用途 | 推奨スタイル |
|------|------------|
| 動画生成まで完走させたい場合 | `anime_japanese_korean`（日韓アニメ） |
| 雰囲気重視・試作 | `anime_ghibli`、`3D_CG_Animation` |
| 実写風（静止画のみ） | `photorealistic_real_human` |

> **注意：** `photorealistic_real_human` スタイルの画像は、動画生成モデル（Seedance 2.0、Kling O3等）のセーフティフィルターにより動画化がブロックされる場合がある。動画生成まで完走させる場合はアニメ系スタイルを優先すること。

### 3-2. 動画モデル選定

| モデル | 特徴 | 推奨用途 |
|--------|------|---------|
| `generate_video_by_kling_o3` | アニメ・CG系に強い | アニメスタイルの動画生成 |
| `generate_video_by_seedance_20` | 高品質・汎用 | 試作以外の本番制作 |
| `generate_video_by_kling_v3_0` | 安定動作 | バックアップ |

### 3-3. アスペクト比

| 用途 | 設定 |
|------|------|
| YouTube通常動画 | `16:9` |
| YouTube Shorts / TikTok / Reels | `9:16` |
| Instagram正方形 | `1:1` |
