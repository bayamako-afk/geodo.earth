# EP001 FlexClip 編集検証手順

## 1. 目的

『職場ウイルス図鑑』Season 1 EP001「ため息ウイルス」において、ZopiaとFlexClipの分業体制を検証するための手順書。

## 2. 検証フロー

1. Zopiaで本編映像をBGMなし、または仮音付きで出力する。
2. FlexClipにMP4を読み込む。
3. 冒頭にv2の共通OPを配置する。
4. 本編ではBGMを薄くする、または無音寄りに調整する。
5. ため息や沈黙の間を邪魔しないよう音量調整を行う。
6. ラストで「ため息ウイルス」のテキストを表示する。
7. 「その違和感に名前をつけろ」＋ロゴを表示する。
8. v2を短くフェードアウトさせる。
9. 15秒版 / 30秒版 / 60秒版を書き出し比較する。

## 3. 検証ポイント

- v2のOPが重すぎず、職場ドラマの入口として機能するか。
- 本編中のため息・沈黙をBGMが邪魔しないか。
- ロゴ表示と主題歌のピークが合うか。
- 15秒版でも意味が伝わるか。
- 30秒版でシリーズ感が出るか。
- 60秒版で余韻が出るか。
