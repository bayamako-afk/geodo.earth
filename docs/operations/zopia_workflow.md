# Zopia 映像生成ワークフロー標準

## 1. 目的

『職場ウイルス図鑑』Season 1 において、Zopiaで担当する映像生成の役割と、FlexClipへ引き継ぐための出力形式を定義する。

## 2. 役割分担

Zopiaは、実写ショートドラマ風の映像生成、登場人物の芝居・表情・間の演出、AIセリフ・ナレーションの生成を担当する。音源の最終調整はFlexClipで行うため、Zopia側での音声編集は最小限に留める。

## 3. Zopiaでの作業内容

Zopiaでは以下の作業を行う。

- 実写ショートドラマ風の映像生成
- 登場人物の芝居・表情・間の演出
- AIセリフまたはナレーションの生成
- 各話ごとの「職場ウイルス」演出

## 4. 出力仕様

Zopiaからの出力は、FlexClipで扱いやすい形式とする。

- **フォーマット**: MP4
- **音声**: BGMなし、または仮BGM程度の出力とする。
- **解像度**: （※必要に応じて規定）

## 5. 注意点

- Zopiaでの音声編集（BGM開始位置の精密制御、任意尺切り出し、フェード、個別音量調整など）は量産運用に不安があるため、深追いしないこと。
- 映像とセリフの同期に注力し、BGMやSEは後工程（FlexClip）に任せる前提で映像を構成する。
