# FlexClip 編集ワークフロー標準

## 1. 目的

『職場ウイルス図鑑』Season 1 において、Zopiaで生成した映像素材に対し、主題歌・字幕・ロゴ・効果音をFlexClipで組み込み、最終的な映像作品として完成させるための標準手順を定める。

## 2. 前提

Zopia側で映像・AIセリフの生成を行い、FlexClip側で音源の最終調整と字幕・ロゴの追加を行う分業体制とする。

## 3. FlexClip テンプレート構成

### 3.1 共通OPテンプレート

Season 1 の各話共通のオープニングとして使用する。

- **使用音源**: `assets/災難は突然にv2.mp3`
- **構成案** (15秒以内):
  - 0:00〜0:02: 短い不穏な導入
  - 0:02〜0:06: 主題歌 v2 の入り
  - 0:06〜0:10: 職場の違和感カット
  - 0:10〜0:13: エピソードタイトル表示
  - 0:13〜0:15: 職場ウイルス図鑑ロゴ

### 3.2 共通EDテンプレート

各話のエンディングとして使用する。

- **構成案**:
  - 職場ウイルス名表示
  - 短い余韻
  - 「その違和感に名前をつけろ」テキスト表示
  - 職場ウイルス図鑑ロゴ表示
  - 主題歌 v2 フェードアウト
- **使用音源**: v2の終盤またはサビ後半

### 3.3 告知PVテンプレート

ティザーやSNS投稿用の短尺動画として使用する。

- **使用音源**: `assets/災難は突然にv1.mp3`
- **尺**: 15秒 / 30秒版
- **テキスト案**: 「近日公開」「感染拡大中」など

## 4. 音量調整基準

FlexClipでの音声ミックスにおける音量バランスの目安は以下の通り。

| 要素 | 音量方針 |
| --- | --- |
| セリフ | 最優先で聞こえるよう調整 |
| 主題歌OP | やや大きめ |
| 本編中BGM | セリフを邪魔しない程度に抑える |
| SE | 必要最低限に留める |
| ED主題歌 | ロゴ表示に合わせて少し強め |
| フェード | 急に切らず自然にフェードイン/アウト処理 |

## 5. 編集時の注意点

- Zopia出力のMP4をベースに作業を行うこと。
- セリフの聞き取りやすさを最優先とし、BGMが被る場合はBGMの音量を下げる。
- 各話ごとにゼロから編集するのではなく、上記のテンプレートを活用して効率化を図る。
- 公開用ファイルと作業用ファイルは明確に分けて管理する。
