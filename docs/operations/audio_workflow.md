# 音源運用ワークフロー標準

## 1. 目的

『職場ウイルス図鑑』Season 1 における、主題歌『災難は突然に』v1 / v2 の運用ルールを定める。

## 2. 使用音源

- `assets/災難は突然にv1.mp3`
- `assets/災難は突然にv2.mp3`

## 3. 音源の使い分けルール

| 用途 | 推奨音源 | 理由 |
| --- | --- | --- |
| EP001〜EP005 共通OP | v2 | シリーズの標準主題歌として安定 |
| 各話ED | v2 | 違和感・余韻・命名に合う |
| 公開前ティザー | v1 | 勢いがあり、告知向き |
| SNS用短尺PV | v1 | テンポ感が強く、目を引きやすい |
| コメディ色が強い回 | v1も検討 | 明るい皮肉として機能 |
| 不穏さ・孤独感が強い回 | v2 | Season 1の空気に合う |

## 4. 音源運用の基本方針

- v2をSeason 1の共通OP・標準主題歌とし、ブランド統一を図る。
- v1は捨てずに、告知・PV・テンション高め用途で活用する。
- 既存有名曲に似せない運用を維持する。
- Zopiaで映像を生成し、FlexClipで音源・字幕・ロゴを組み込む。
