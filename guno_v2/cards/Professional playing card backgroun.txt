Professional playing card background for a competitive strategy card game called GUNO™ (Tokyo Edition).

Portrait orientation.
Final print-ready size: 69mm x 94mm (includes 3mm bleed for 63mm x 88mm trim).
High resolution, 300dpi (approx 816 x 1110 pixels).
RGB color space.
Clean flat design.

Main color: JR Yamanote Line green (#00AA00).

Outer border:
Solid thick frame using the exact main route color.
Border thickness must be 6% of card width.
Corner radius small and crisp (approx 4.5mm).
No shadow.

Inside the border:
Add a thin white inner line (premium finish), about 1.5% of card width.

Inner panel:
Very light off-white surface (#F4F6F3).
Balanced centered geometry.
No gradient.

Background motif:
Very subtle circular transit ring centered on card (opacity 6%).
Light abstract circuit network pattern (opacity 8%).
Low contrast.
Identical layout structure across all color versions.

Important:
No numbers.
No text.
No logos.
No corner panels.
No decorative shapes.
No glow.
No neon.
No metallic effects.
No dramatic lighting.
No heavy gradients.

Modern competitive card game aesthetic.
Prepared for HTML overlay (numbers and station names will be added digitally).
