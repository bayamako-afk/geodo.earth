# GUNO V6

GUNO V6は、オンライン対戦機能を備えたGUNOの次世代バージョンです。

## 目的と位置づけ

既存の `guno_v5` は「ローカル完成版」「ルール検証・公開デモ版」として維持されます。
本ディレクトリ `guno_v6` は、以下の目的のために**完全に独立した基盤**として新規構築されています。

- **オンライン対応の本命基盤** (Supabase等を利用したリアルタイム同期)
- **GUNO Pack v1.0 標準対応**
- **将来的な拡張（世界制覇ログ、ランキング等）の土台**

**※重要方針※**
V6の開発によって、既存の `guno_v5` のコードに破壊的変更やオンライン機能の直接継ぎ足しを行うことはありません。

## 現在のステータス

**現在、ローカル最小実装中（基盤構築フェーズ）です。**
段階的に、以下の順序で実装を進めています。

1. 基盤構築 (v6-foundation)
2. Pack v1.0 ローダー (v6-pack-loader)
3. ピュアゲームエンジン分離 (v6-core-engine)
4. スコアリング分離 (v6-scoring)
5. ローカル最小UI (v6-local-ui)
6. オンラインルーム基盤 (v6-online-room / v6-supabase-sync)

## ディレクトリ構成

```text
guno_v6/
  src/
    app/      # エントリーポイント、ルーティング
    core/     # UI非依存の純粋なゲームエンジン、ルール、状態管理
    data/     # Packローダー、データ管理
    ui/       # 描画コンポーネント（Board, Hand, Log等）
    net/      # 通信、状態同期（Supabase等）
    shared/   # 定数、共通ユーティリティ
```
